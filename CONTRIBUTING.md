# Contributing to dose-notes

## Clone

```bash
git clone https://github.com/hinanohart/dose-notes.git
cd dose-notes
```

## Making changes

Edit files directly. No build step required.

## Branch convention

Use `feat/<topic>`, `fix/<topic>`, or `chore/<topic>` branches off `main`.

## Pull request convention

- One logical change per PR.
- Keep additions focused and concise.
