# dose-notes

Critical Theory companion to [dose](https://github.com/hinanohart/dose).

Draft essays and philosophical commentary land here, NEVER in the `dose/` core library.

The `dose` core is intentionally free of narrative — only code, metrics, and results.
All interpretive framing, philosophical grounding, and critical theory discussion lives in this repository.

## Contents

- `essays/` — Draft essays and critical commentary
- `essays/_template.md` — Template for new essays

## Scope

This repository is for:
- Philosophical framing of the dose experiment
- Critical theory essays related to dual-use evaluation
- Draft writings for academic or public venues
- Interpretive commentary on PSI and DRC results

This repository is NOT for:
- Python code (belongs in `dose`)
- Experimental data or metrics (belongs in `dose/reports/`)
- Release artifacts
